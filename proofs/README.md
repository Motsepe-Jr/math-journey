## Proof Techniques Learned

1. **Existence Proof** - Show one example to prove "there exists"
2. **Parity Argument** - Use even/odd invariants
3. **Coloring Argument** - Find hidden constraints via labeling/coloring
4. **Invariants** - Properties that must be preserved across valid configurations
5. **Pigeonhole Principle** - If kn+1 objects go into n containers, at least one container has k+1 objects
