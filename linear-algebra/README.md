# Linear Algebra

Working through **Linear Algebra Done Right** by Sheldon Axler (4th Edition, 2024).

## Why This Book

Axler's approach is different from most intro linear algebra courses — instead of starting with matrices and determinants, it builds everything from vector spaces up. The focus is on understanding structure, not just computation. It's more abstract but the proofs feel cleaner because of it.
